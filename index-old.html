1. 個人情報の収集

当アプリは、氏名、メールアドレス、位置情報、その他ユーザーを識別できる個人情報を 一切収集・送信しません。

4. 広告

当アプリには、サードパーティ広告 SDK や広告配信機能は含まれていません。

5. お問い合わせ

English

This policy explains how "OmniPaneExplorer" (Productivity / Utility), distributed on Microsoft Store, handles personal information and user data.

1. Personal Information Collection

The app does not collect or transmit personal information such as names, email addresses, location data, or any other personally identifiable information.

2. File Access and Data Processing

With explicit user permission, the app accesses local files and SharePoint files. Core features including the 4-pane explorer, SharePoint integration, profile duplication, and regex replacement are processed locally on the user's device. No file content is sent to developer-operated or third-party external servers.

3. Payments

In-app purchases (such as the Professional edition) are processed through the Microsoft Store payment system. The developer does not receive or store credit card details.

4. Advertising

The app does not include third-party advertising.

5. Contact

For inquiries, please use the developer contact channels provided on Microsoft Store.

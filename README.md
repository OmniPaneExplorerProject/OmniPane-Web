# OmniPane Explorer (OPE) 🚀

**ファイル管理、テキスト編集、ターミナルを効率的に集約する統合型ワークスペース**

Windows標準のエクスプローラーで、複数のウィンドウを開いて画面が煩雑になっていませんか？  
フォルダの階層が深すぎて、目的のファイルにアクセスするまでに手間がかかっていませんか？

**OmniPane Explorer** は、ファイル操作、テキスト編集、コマンドライン実行を1つの画面に集約し、独自のメタデータ管理によって効率的なファイル整理をサポートする、Windows専用の「統合型ワークスペース・ファイラー」です。

---

## 🧭 OPEが日々の作業を効率化する、4つのアプローチ

### 1. すべてが1画面に閉じる「マルチペイン・タブ構造」

ウィンドウ切り替えによる視線の移動や集中力の切断を和らげます。

- **マルチペイン構成**: 独立した画面分割により、フォルダ間のファイル参照や移動、コピー作業をスムーズに行えます。
- **仮想パスの融合**: 通常のフォルダだけでなく、内蔵テキストエディタ（`text://`）やターミナル（`terminal://`）を同じ画面の「タブ」として並列で配置可能。仕様書を確認しながらコードを編集し、そのままコマンドを実行する、といった一連の作業がシームレスに完結します。

### 2. フォルダ階層だけに頼らない「タグ ＆ メモ機能」

「どのフォルダに格納すべきか」という分類の悩みから解放されます。

- ファイルやフォルダに対して、個別に**独自のタグ、メモ、評価（スター）**を付与できます。
- 案件名や進捗ステータス、重要度など、用途に応じた柔軟な条件での並び替えや管理が可能です。

### 3. ファイルの移動やリネームに配慮した「PathFollow構造」

「ファイル名を変えたり移動したりすると、付与した情報を見失ってしまう」という問題に対応しています。

- WindowsのFRN（File Reference Number）構造を活用したシステム設計。
- アプリケーション内でファイル名が変更されたり、別フォルダへ移動されたりした場合でも、**付与したタグやメモ情報が裏側で追従・維持されやすい仕組み**を備えています。

### 4. 柔軟な条件でファイルを炙り出す「ハイブリッド検索」

「あの案件に関連する、重要度が高く、メモに特定の文言を残したテキストファイル」といった、記憶を頼りにした検索をサポートします。

- 名前・タグ・メモ・ファイル内文字列（Grep）を組み合わせた検索エンジンを搭載。
- 蓄積されたデータから、目的のファイルを効率的に引き当てることができます。

## 🛠️ エディション ＆ クォータ（制限）について

OmniPane Explorerは、無料の **Community版** から基本機能をお試しいただけます。日常の業務に溶け込み、より高度な多画面レイアウトや自動化を行いたくなったタイミングで、最適なプランへアプリ内からアップグレード可能です。

| 機能 / エディション | Community (無料版) | Standard | Professional (最上位) |
| :--- | :---: | :---: | :---: |
| **同時表示ペイン数** | **最大 2 ペイン** <br> *(左右 / 上下分割)* | **最大 4 ペイン** <br> *(4画面フル開放)* | **最大 4 ペイン** <br> *(4画面フル開放)* |
| **1ペインあたりのタブ数** | 最大 3 タブ | 最大 8 タブ | **無制限** |
| **タグ・メモ付与・検索** | 件数クォータ制 <br> *(タグ15件/メモ10件まで)* | **無制限** | **無制限** |
| **正規表現の一括置換 (Grep/Replace)** | - | - | **○ (完全開放)** |
| **プロファイル複製 (レプリケーション)** | - | 簡易版 | **○ (フル機能)** |

### 🎁 まずは無料で、この快適さを体験してください

無料のCommunity版でも、2ペインの中にエディタやターミナルを組み込む便利さや、パス追従（PathFollow）の利便性は期間制限なしで体感いただけます。  
まずはご自身のファイル整理に「タグやメモ」を取り入れる快適さを体験してください。その便利さをさらに拡張したくなったとき、4ペインの広大な作業領域があなたのビジネスを強力にバックアップします。

## SEO

- `robots.txt` / `sitemap.xml` を配置済み（`demo.html` は video sitemap 付き）
- 各 HTML に canonical・Open Graph・Twitter Card を設定
- トップに SoftwareApplication の JSON-LD、`demo.html` に VideoObject を設置
- 動画インデックス用の再生ページ: `demo.html`（トップの埋め込みは補足コンテンツ）
